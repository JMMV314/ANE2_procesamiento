Convierte un arreglo de datos en formato C\+S8 a un vector de números complejos IQ. 

Esta función toma un arreglo de datos de 8 bits y los convierte en números complejos representados como {\ttfamily complex double}. Los datos se interpretan como pares consecutivos de valores, uno para la parte I (In-\/phase) y otro para la parte Q (Quadrature).


\begin{DoxyParams}{Parámetros}
{\em raw\+Vector} & Un puntero al arreglo de datos en formato C\+S8. \\
\hline
{\em length} & La longitud del arreglo {\ttfamily raw\+Vector}. \\
\hline
{\em num\+\_\+samples} & Puntero donde se almacenará el número de muestras convertidas a números complejos.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Devuelve}
Un puntero a un arreglo de números complejos {\ttfamily complex double}, representando el vector IQ.
\end{DoxyReturn}
\begin{DoxyNote}{Nota}
La longitud del arreglo debe ser par, ya que cada par de valores será interpretado como una muestra compleja. 
\end{DoxyNote}

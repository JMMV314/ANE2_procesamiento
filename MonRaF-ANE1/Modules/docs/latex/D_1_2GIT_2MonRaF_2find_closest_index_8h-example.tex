Encuentra el índice del elemento en el arreglo que está más cerca de un valor dado.\+Esta función recorre un arreglo de números de punto flotante ({\ttfamily double}) y calcula la diferencia absoluta entre cada elemento y el valor objetivo {\ttfamily value}. Retorna el índice del elemento cuya diferencia es la menor.


\begin{DoxyParams}{Parámetros}
{\em array} & Un puntero al arreglo de valores de tipo {\ttfamily double}. \\
\hline
{\em length} & La cantidad de elementos en el arreglo. \\
\hline
{\em value} & El valor al cual se desea encontrar el elemento más cercano.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Devuelve}
El índice del elemento en el arreglo que es más cercano al valor dado {\ttfamily value}.
\end{DoxyReturn}
\begin{DoxyNote}{Nota}
Si el arreglo contiene varios elementos a la misma distancia de {\ttfamily value}, la función retorna el índice del primer elemento encontrado.
\end{DoxyNote}

Descripción detallada

Programa principal para la manipulación de señales IQ en formato CS8.

Este programa realiza varias operaciones relacionadas con señales IQ, incluyendo:

  • Carga de bandas de frecuencia y parámetros de canalización.
  • Lectura de datos en formato CS8 desde un archivo.
  • Conversión de datos binarios a vectores IQ complejos.

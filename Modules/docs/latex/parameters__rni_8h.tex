\doxysubsection{Descripción detallada}
Definción de funciones que calculan parámetros para niveles de radiación electromagnética (R\+NI). 

Este archivo contiene la definición de la función {\ttfamily parameter\+\_\+rni}, que analiza señales IQ para calcular parámetros relacionados con radiación electromagnética, incluyendo densidad espectral de potencia (P\+SD), intensidad de campo eléctrico (V/m) y límite ocupado.

Realiza análisis espectral y calcula parámetros relacionados con radiación electromagnética. 

Esta función procesa una señal IQ para calcular su P\+SD mediante el método de Welch, analiza canales definidos, y genera parámetros como intensidad de campo eléctrico (V/m) y porcentaje del límite ocupado.

\doxysubsection{Descripción detallada}
Función que calcula la Densidad Espectral de Potencia de un array complejo y genera sus frecuencias asociadas. 

Esta función permite almacenar datos de frecuencias y densidades espectrales de potencia (P\+SD) en un archivo C\+SV con formato adecuado para análisis posterior.

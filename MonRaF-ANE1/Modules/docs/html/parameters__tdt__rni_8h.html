Descripción detallada

Definción de función para analizar señales RNI y generar un archivo JSON con resultados.

Este archivo contiene la definición de la función parameter_tdt_rni que calcula métricas relacionadas con la radiación electromagnética, como la intensidad de campo eléctrico (V/m) y el porcentaje del límite ocupado, guardándolos en un archivo JSON.

Analiza datos de señales RNI y guarda los resultados en un archivo JSON.

Parámetros
modulationEsquema de modulación utilizado (por ejemplo, QAM, PSK).
central_freqFrecuencia central de la señal (en Hz).
file_sampleIdentificador del archivo de muestra CS8 que contiene los datos IQ.

La función procesa datos IQ para calcular métricas como:

  • Intensidad del campo eléctrico ( V/m ).
  • Porcentaje del límite permitido para la intensidad del campo.

Los resultados se almacenan en un archivo JSON llamado parameters_tdt.json.

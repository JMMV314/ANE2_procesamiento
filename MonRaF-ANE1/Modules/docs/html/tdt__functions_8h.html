Descripción detallada

Definición de funciones para el análisis y procesamiento de señales IQ para parámetros clave.

Este archivo incluye funciones para calcular parámetros como C/N, MER, y BER usando técnicas de procesamiento de señales, como el método de Welch, y genera resultados relacionados con modulación digital.

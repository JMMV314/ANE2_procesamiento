Descripción detallada

Definición de funciones que calcula la Densidad Espectral de Potencia de un array complejo y genera sus frecuencias asociadas.

Esta función permite almacenar datos de frecuencias y densidades espectrales de potencia (PSD) en un archivo CSV con formato adecuado para análisis posterior.

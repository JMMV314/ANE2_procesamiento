\doxysubsection{Descripción detallada}
Programa principal para la manipulación de señales IQ en formato C\+S8. 

Este programa realiza varias operaciones relacionadas con señales IQ, incluyendo\+:
\begin{DoxyItemize}
\item Carga de bandas de frecuencia y parámetros de canalización.
\item Lectura de datos en formato C\+S8 desde un archivo.
\item Conversión de datos binarios a vectores IQ complejos. 
\end{DoxyItemize}

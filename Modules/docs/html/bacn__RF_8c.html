Descripción detallada

Implementación de captura de datos IQ con HackRF.

Este archivo contiene la implementación de funciones para manejar la captura de datos IQ utilizando un dispositivo HackRF. Incluye manejo de señales, configuración del dispositivo y almacenamiento de datos en archivos binarios.
